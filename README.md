# 3rdJCG_Bot


## Getting Started

1. Download Zip Archive and Extract
2. Fill out config.json with API Token, Bot Roles, Mission Path, and DB Path
3. Launch launch.batch

### Prerequisites

Microsoft SQL Community 2019 (or later)
